Mes commandes GIT

Création de branche git branch NomDeLaBranche git checkout NomDeLaBranche git add . git commit -m git push OU git push --set-upstream origin NomDeLaBranch pour créer lien entre branche local et branche distante

Récupérer une branche d'un dépot distant git branch git branch -a git checkout NomBranche

Copier un changement d'une branche vers master (cherry pick) => BugFix urgent à appliquer sur master git checkout master git log / git lock NomBranche pour voir les SHA1 de commit git cherry-pick SHA1 du commit git push

Merge entre deux branches On se positionne d'abord sur la branche sur laquelle on veut faire le merge => ex : git checkout/switch master git merge NomDeLaBrancheAMergerVersMaster git add . git commit -m"Merge de ma branche X" git push

Rebase entre deux branches On se positionne sur la branche que l'on veut ramener vers l'autre ou sur un commit git checkout -b NomDeBrancheARamener git add . git commit git rebase master Si conflit , gérer conflit => git add . => git rebase --continue git checkout master git merge NomDeBrancheARamener

Supprimer une branche git branch BRANCH_A_SUP git checkout BRANCH_A_SUP git push --set-upstream origin BRANCH_A_SUP
