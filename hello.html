Mes récompenses

Mes commandes GIT

Création de branche
  • git branch NomDeLaBranche
  • git checkout NomDeLaBranche
  • git add .
  • git commit -m
  • git push OU git push --set-upstream origin NomDeLaBranch pour créer lien entre branche local et branche distante
Récupérer une branche d'un dépot distant
  • git branch
  • git branch -a
  • git checkout NomBranche
Copier un changement (commit) d'une branche vers master (cherry pick) => BugFix urgent à appliquer sur master
  • git checkout master
  • git log / git lock NomBranche pour voir les SHA1 de commit
  • git cherry-pick SHA1 du commit
  • git push
Merge entre deux branches

On se positionne d'abord sur la branche sur laquelle on veut faire le merge => ex : git checkout/switch master

  • git merge NomDeLaBrancheAMergerVersMaster
  • git add .
  • git commit -m"Merge de ma branche X"
  • git push
Rebase entre deux branches

On se positionne sur la branche que l'on veut ramener vers l'autre ou sur un commit

  • git checkout -b NomDeBrancheARamener
  • git add .
  • git commit
  • git rebase master
    • Si conflit , gérer conflit => git add . => git rebase --continue
  • git checkout master
  • git merge NomDeBrancheARamener
Supprimer une branche
  • git branch BRANCH_A_SUP
  • git checkout BRANCH_A_SUP
  • git push --set-upstream origin BRANCH_A_SUP
Gérer conflit avec stash

Problème de merge : parfois quand on a un fichier local et qu'on fait un pull qui rapatrie le même fichier modifier en conflit avec notrze modification , il faut faire soit un commit de changment , soit un stash ,pull

  • Git stash save "Message"
  • git stash pop index
  • git stash list
  • git stash show index
Envoyer nos modifications local vers le serveur distant
  • git add .
  • git commit -m""
  • git remote add origin https://github.com/Manug1979/WebPageForGit.git => pour faire lien entre dépot local et repo distant Github
  • git push -u origin master => Pousser dépot local vers depot distant
  • git push --tags
Récupèrer les modifications serveur pour les ramener localement
  • Git pull
  • Git fetch
Autres commandes utiles
  • Git config --global user.name => récupérer le nom d'utilisateur
  • Git config --global user.email => récupérer l'email'
  • Git config --global --list
  • git = git --help => demande aide
  • Git version => récupérer version
  • Pwd => Path courant
  • Git init => pour créer un dépot local -> génère dossier .git qu'il ne faut pas effacer car contient l'historique des modifications
  • ls -la => pour vérifier que le dépot a été créé
  • Git add index.html => ajouter fichier à la zone d'index
  • Git reset index.html => retirer fichier de la zone d'index
  • Git add . => permet d'ajouter tous les fichiers 'untracked' vers la zone d'index
  • Git commit -m "NomDuBugOuDeLaFeature"
  • Git diff => voir les modifications
  • Git diff --cached => voir les modifications présentes dans la zone d'index
  • Git log => historique des commit
  • Git show SHA-1 (à copier coller)
  • Git show TagName ( ex : git show master)
  • git clone url https ou lien ssh
  • git remote -v => affiche la liste des remote
  • git merge --abort
  • git pull --rebase
  • git push --set-upstream origin NomDeLaBranch
